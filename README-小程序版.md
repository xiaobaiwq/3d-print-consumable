# 3D打印耗材库存管理系统小程序版

## 导入方式

1. 打开微信开发者工具。
2. 选择“导入项目”。
3. 项目目录选择当前目录。
4. 当前仓库已将 `project.config.json` 中的 `AppID` 脱敏为 `touristappid`，适合公开仓库使用。
5. 如果需要绑定你自己的小程序，请只在本机填写真实 `AppID`，不要把真实值提交回仓库。
6. 本地私有配置建议保存在 `project.private.config.json` 中，该文件已在 `.gitignore` 中忽略。
7. 导入后直接编译即可。

## 已实现能力

- 本地库存列表展示
- 搜索、品牌筛选、系列筛选、排序
- 新增、编辑、删除
- 图片选择与本地缓存
- 本地统计卡片
- JSON 备份与恢复
- CSV 导入与导出

## 说明

- 数据保存在小程序本机缓存中。
- 备份 JSON 和导出 CSV 会写入小程序沙盒目录，并复制文件路径到剪贴板。
- 当前版本没有接入 `xlsx` 解析库，因此保留了 CSV 导入导出，没有直接做 `.xlsx` 文件解析。
